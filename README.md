# AI Language Translator

## Overview

AI Language Translator is a web application that translates text between multiple languages using a translation API. It also provides text-to-speech functionality for improved accessibility.

## Features

* Translate text between multiple languages
* User-friendly interface
* Copy translated text
* Text-to-Speech support
* Real-time translation using API integration

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* translate-google package

## Installation

1. Clone the repository
2. Install dependencies:

npm install

3. Start the server:

node server.js

4. Open index.html using Live Server

## Languages Supported

* English
* Tamil
* Hindi
* French
